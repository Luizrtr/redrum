# Redrum APP 🎲
> This is a repository for services management with dashboard realtime.

## Getting Started🧑🏾‍💻

First, you need the ".env" file with environment variables:

```bash
HOST=locahost:3000
```

after, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open ```HOST``` with your browser to see the result.
